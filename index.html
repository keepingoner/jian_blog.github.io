Git

发表于 2018-04-03

Git 常用命令大全

git 初步运行配置

  1. git config –global user.name “John Doe”
  2. git config –global user.email johndoe@example.com

git 查看配置信息

git congfig –list

初始化仓库

git init

添加跟踪文件

git add

提交

git commit -m “描述”

推送

git push origin master

克隆现有仓库

git clone url

查看文件状态

git status
git status -s （状态简览）

写忽略文件

cat .gitignore

  • .[oa]

删除文件

git rm 文件

git rm -r 文件夹

git diff

  1. git diff 比较的是工作区和暂存区的差别
  2. git diff –cached 比较的是暂存区和版本库的差别
  3. git diff HEAD 可以查看工作区和版本库的差别

查看提交历史

git log

git log -p （显示每次提交的差异）

git log –stat （查看提交统计信息）

撤销操作

git commit –amend

git reset HEAD 文件
