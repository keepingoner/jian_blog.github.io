Git

发表于 2018-04-03

git 初步运行配置

git config –global user.name “John Doe”
git config –global user.email johndoe@example.com

git 查看配置信息

git congfig –list

初始化仓库

git init

添加跟踪文件

git add

提交

git commit -m “描述”

推送

git push origin master

克隆现有仓库

git clone url

查看文件状态

git status
git status -s （状态简览）

写忽略文件

cat .gitignore

* .[oa]

删除文件

git rm 文件

git rm -r 文件夹

git diff

git diff 比较的是工作区和暂存区的差别

git diff –cached 比较的是暂存区和版本库的差别

git diff HEAD 可以查看工作区和版本库的差别

查看提交历史

git log

git log -p （显示每次提交的差异）

git log –stat （查看提交统计信息）

撤销操作

git commit –amend

git reset HEAD 文件

git checkout 文件 慎用

远程仓库

git remote -v (显示远程仓库)

git remote show origin

添加远程仓库

git remote add shortname

移除远程仓库

git remote rename old new

git remote rm name

git 打标签

git tag

git tag -l “”

查看标签和对应信息

git show 标签

检出标签

git checkout -b [branchname] [tagname]
